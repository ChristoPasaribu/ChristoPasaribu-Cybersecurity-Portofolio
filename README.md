## 📘 Tentang Portofolio
Portofolio ini merupakan dokumentasi pembelajaran dan praktik baik dari sisi **offensive security** maupun **defensive security**.  
Seluruh aktivitas eksploitasi dilakukan pada **lingkungan lab pribadi**, VM lokal, atau target yang memang mengizinkan.

Tujuan portofolio:
- Memahami alur serangan  
- Membaca dan menganalisis log  
- Mengidentifikasi ancaman  
- Memberikan rekomendasi mitigasi  
- Membuat laporan profesional
